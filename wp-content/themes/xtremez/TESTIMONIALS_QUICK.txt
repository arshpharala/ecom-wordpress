## ğŸ”§ Helper Functions

### Get Testimonials
```php
// Get all (use default menu_order)
$testimonials = xtremez_get_testimonials();

// Get 6 testimonials
$testimonials = xtremez_get_testimonials(6);

// Get 5 by date (newest first)
$testimonials = xtremez_get_testimonials(5, 'date', 'DESC');

// Get 3 by menu order (for custom ordering)
```

## ğŸ“ Escaping Output

Always escape when displaying:

```php
// For HTML content
echo wp_kses_post($data['text']);

// For plain text
echo esc_html($data['name']);

// For URLs (if logo is used as link)
echo esc_url($data['logo']);

// For HTML attributes
echo esc_attr($data['position']);
```
